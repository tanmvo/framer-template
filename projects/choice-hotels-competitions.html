Choice Hotels Competitions

Client: Choice Hotels

ux ui user flows wireframes photoshop wordpress

Choice Hotels approached Taos Creative to create a microsite to host their online competitions. Separate competitions were held in Australia and New Zealand, so the requirements were to create a template where elements could be easily updated. The competitions were exclusive to Choice Hotels customers, therefore the client needed a secure gateway for users to log in.

My Role

  • Define user goals and tasks, balancing their needs with business requirements
  • Produce user flows to map out journey
  • Create wireframes to demonstrate functionality
  • Lead team of developers to implement responsive microsite
  • Manage multiple stakeholders, ensuring business requirements were accomplished within a strict timeframe

Solution

We designed a process where users can enter their unique ID which will grant access and automatically fill in the competition form. To entice users to subscribe to the company's Facebook page, we added a facebook plugin so entrants could "like" the page and receive additional rewards. The prompt loads on form completion, giving the user a choice to subscribe to the Facebook fan page.

Wireframes and userflows were originally built using protoshare, a cloud based wireframing tool.

User flow for Choice Hotels microsite

Result

The microsite registered 1000+ new Choice Hotels members and 5000+ Facebook likes.
